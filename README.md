## Deploying locally
First of all, you will need to fork this project:
```bash
git clone https://github.com/d1loop/docs.git
```
After this, you'll need to run:
```bash
yarn
```
Or if you are using NPM:
```bash
npm install
```
After that, run:
```bash
yarn start
```
Or if you are using NPM:
```bash
npm run start
```

## Building
To build this, you'll need to run:
```bash
yarn build
```
Or if you are using NPM:
```bash
npm run build
```
